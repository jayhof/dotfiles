# Dotfiles

My dotfiles for the macOS platform.

## Included

- developer tools
- config files
- preferences

## Setup

- `$ git clone [this repo]`
- `$ sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"`
- `$ ./install.sh`
- Open iTerm Settings/General/Preferences and load from ~/dotfiles
